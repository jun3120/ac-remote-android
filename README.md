# 空调遥控器 (AC Remote Android)

Android 原生红外空调遥控器 APP，通过手机红外发射器控制空调。

## 功能规划

- 开关控制
- 温度调节
- 模式切换（制冷/制热/送风/除湿）
- 风速调节
- 品牌适配

## 技术栈

- **语言**: Kotlin
- **最低 SDK**: Android 7.0 (API 24)
- **红外**: ConsumerIrManager
- **UI**: Material Design 3 + ConstraintLayout

## 开发环境

- Android Studio Hedgehog+
- JDK 17
- Gradle 8.x + AGP 8.5
